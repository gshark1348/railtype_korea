## 검증 기준

1. 모든 역사 좌표가 대한민국 경계 폴리곤 내부 또는 경계 위에 있는지 검사
2. 역사 순서대로 연결한 대표 중심선이 국토 밖으로 벗어나는지 검사
3. 첫 역과 마지막 역, 양방향 배열, 역사 코드와 영문명을 확인
4. 런타임 JavaScript 데이터와 CSV의 역사 수·순서 일치 여부 확인
