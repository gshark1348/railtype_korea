# RAILTYPE KOREA v27.1 · 수인분당선 GIS 위치 검증

## 수정 원인

수인분당선의 `COURSE_REGION_IDS`가 도시 ID 문자열이 아닌 역 구간 객체 배열로 등록되어 있었습니다. 지도 렌더러는 도시 ID를 키로 폴리곤을 찾으므로 서울·성남·수원·화성·안산·시흥·인천의 기존 경계가 모두 누락되었습니다. 또한 기존 경계 파일에는 용인시 피처가 없었습니다.

## 보정 내용

- 수인분당선 렌더링 도시 ID를 `seoul → seongnam → yongin → suwon → hwaseong → ansan → siheung → incheon` 순서로 수정
- 용인시 수지구·기흥구·처인구 폴리곤을 dissolve하여 단일 용인시 외곽 경계 생성
- 모든 도시 경계와 63개 역을 WGS84 / EPSG:4326으로 통일
- 화면용 역 위치 오프셋 제거 상태 확인: 역 노드·노선·열차가 동일 좌표 사용
- 거리 검증은 UTM-K / EPSG:5179로 수행

## 참고

역 좌표는 역사 건물 전체의 면적이 아니라 공공 철도 데이터에서 제공하는 단일 기준점입니다. 노선 선형은 역 기준점을 순서대로 연결한 교육용 인포그래픽이며 실제 궤도 중심선 측량 데이터는 아닙니다.
