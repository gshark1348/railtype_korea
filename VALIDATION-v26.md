# RAILTYPE KOREA v26.0 · KTX 호남고속축 GIS 검증

## 검증 대상

- 코스: **용산 → 광명 → 천안아산 → 오송 → 공주 → 익산 → 정읍 → 광주송정 → 나주 → 목포**
- 방향: 목포행 / 용산행
- 역사 수: **10개**
- 좌표계: **WGS84 / EPSG:4326**
- 전국 배경: Natural Earth 행정경계와 GSHHS 중해상도 해안선을 결합한 웹 인포그래픽용 경계

## 결과

| 검증 항목 | 결과 |
|---|---:|
| 대한민국 경계 내부 역사 | **10 / 10** |
| 수면 또는 국토 외부에 놓인 역사 | **0개** |
| 역사 데이터 구조 | **PASS** |
| 목포역 서남해안 포함 여부 | **PASS** |
| 노선별 커뮤니티 분리 키 | **11 · KTX 호남고속선** |

모든 대표 정차역 좌표가 대한민국 경계 내부에 위치합니다.

## 역 간 중심선 해석

화면의 노선은 실제 선로 중심선이나 측량 데이터를 그리는 것이 아니라, 역 좌표를 순서대로 연결한 모션 그래픽입니다. 따라서 하천·하구·교량 접근부를 가로지를 수 있습니다. 이번 검사에서는 다음 구간의 직선 중심선 일부가 수계와 교차했습니다.

- 나주–목포: 국토 폴리곤 외부 중심선 길이(각도 단위) `0.0251506`

이는 **역이 물 위에 배치된 오류가 아니며**, 해당 구간의 실제 철도 선형을 의미하지도 않습니다. 향후 정밀 선로 형상을 제공하려면 국가철도공단 철도 중심선 또는 별도의 경유점 데이터를 적용해야 합니다.

## 호남고속철도 범위 표기

게임 코스는 이용자가 인지하기 쉬운 용산–목포 KTX 호남축을 제공합니다. 다만 전용 고속선 1단계는 오송–광주송정이며, 광주송정 이남은 현재 기존 호남선 연계 구간입니다. 호남고속철도 2단계는 나주·무안국제공항·목포 방향으로 진행되는 사업으로 노선 역사 설명에 구분하여 표시했습니다.
